# Contributing

Thank you for your interest in improving this course!

## How to Contribute

### For Trainers and Educators

- Fix typos or outdated content in any module README
- Add exercises, prompts, or case studies to existing modules
- Improve slide notes or trainer guidance

### For Students

- Share useful resources by adding links to `resources/` folders
- Suggest new project ideas in the `projects/` folder

## Process

1. Fork the repository
2. Create a branch: `git checkout -b my-contribution`
3. Make your changes
4. Commit with a clear message: `git commit -m "Add exercise for Module 04"`
5. Open a Pull Request with a brief description of your changes

## Style Guidelines

- Use plain Markdown for all `.md` files
- Follow the existing module README structure for new modules
- Keep language clear and accessible for college students

## Questions

Open an issue on GitHub or contact the course maintainer directly.
