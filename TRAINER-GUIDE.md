# Trainer Guide

## Before the Course

- Review all module READMEs and slides
- Set up demo accounts: ChatGPT, Microsoft Copilot, GitHub Copilot
- Ensure students have access to required tools
- Share the repo link with students before Week 1

## Delivery Tips

- Each module is designed for a 2-3 hour session
- Start with the slides, then move to live demos, then exercises
- Encourage students to try prompts in real-time during demos
- Allocate at least 30 minutes for hands-on exercises per module

## Module-by-Module Notes

### Module 1 — Introduction to AI
- Use real-world examples students relate to (Netflix, Spotify, Google Maps)
- Poll the class: "What AI tools do you already use?"

### Module 4 — Generative AI & LLMs
- Live demo is essential — show ChatGPT, Copilot, and image generation side by side
- Discuss token limits practically (paste a long article and show truncation)

### Module 6 — Microsoft Copilot
- Students need a Microsoft 365 account with Copilot enabled
- Walk through Word, Excel, and PowerPoint scenarios step by step

### Module 7 — AI for Programming
- Use GitHub Copilot in VS Code for live coding demos
- Pair students for the coding exercises

### Module 8 — Responsible AI
- Use case studies to drive discussion — avoid lecturing
- Invite students to share their own AI ethics concerns

## Final Project

- Announce the project brief at the end of Week 5
- Allow students to pick from the `projects/` folder or propose their own
- Final presentations in Week 10
