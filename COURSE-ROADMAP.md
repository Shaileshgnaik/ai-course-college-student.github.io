# Course Roadmap

## Program Overview

**Duration:** 10 Weeks (1 module per week)
**Level:** Beginner to Intermediate
**Audience:** College Students (any discipline)

---

## Weekly Schedule

| Week | Module | Topics |
|------|--------|--------|
| 1 | Introduction to AI | AI history, types, applications, current landscape |
| 2 | Machine Learning Fundamentals | Supervised, unsupervised, reinforcement learning |
| 3 | Deep Learning & Neural Networks | CNNs, RNNs, transformers |
| 4 | Generative AI & LLMs | GPT, BERT, foundation models, tokens |
| 5 | Prompt Engineering | Prompting techniques, patterns, evaluation |
| 6 | Microsoft Copilot | Word, Excel, PowerPoint, Teams integration |
| 7 | AI for Programming | GitHub Copilot, Python, JavaScript with AI |
| 8 | Responsible AI | Ethics, bias, fairness, safety, governance |
| 9 | Future of AI | Emerging trends, AGI, industry disruption |
| 10 | AI Careers & Roadmap | Certifications, interview prep, career paths |

---

## Learning Path

```
Foundations → Applied AI → Responsible Use → Career Readiness
   (Wk 1-3)     (Wk 4-7)       (Wk 8-9)          (Wk 10)
```

---

## Assessment Structure

- Weekly quizzes (10%)
- Lab exercises (20%)
- Module assignments (30%)
- Final project (40%)
