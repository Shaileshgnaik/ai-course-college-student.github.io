# Lab 04 — Microsoft Copilot

**Duration:** 75 minutes
**Module:** Follows Module 06 — Microsoft Copilot
**Tools needed:** Microsoft 365 with Copilot (or copilot.microsoft.com as fallback)

> **No Microsoft 365 Copilot?** All exercises can be adapted using copilot.microsoft.com — paste text in and out manually.

---

## Objectives

By the end of this lab you will:
- Generate and edit documents using Copilot in Word
- Analyse data and generate insights using Copilot in Excel
- Create a presentation using Copilot in PowerPoint
- Summarise information using Microsoft Copilot

---

## Part 1: Copilot in Word — Document Generation (20 minutes)

**Scenario:** You have been asked to write a 1-page report on "The Impact of AI on the Job Market" for a college newsletter.

**Step 1:** Open Microsoft Word → Click the Copilot icon (or go to copilot.microsoft.com)

**Step 2:** Use this prompt:
```
Draft a 1-page report on "The Impact of AI on the Job Market" for a college student audience.
Structure it with: an introduction, 3 key impacts with brief explanations, and a conclusion.
Keep the tone informative but accessible.
```

**Step 3:** Review the draft and use Copilot to:
- Make the introduction more engaging: *"Rewrite the introduction to be more compelling"*
- Simplify a complex paragraph: *"Simplify this paragraph for a non-technical reader"*
- Add a statistic: *"Add one relevant statistic to support each key point"*

**Document what changed and whether the revisions improved the document.**

## Part 3: Copilot in PowerPoint — Presentation Creation (20 minutes)

**Task:** Create a 5-slide presentation on "AI Tools Every Student Should Know"

**Step 1:** Open PowerPoint → Copilot → *"Create presentation about..."*

**Prompt:**
```
Create a 5-slide presentation titled "AI Tools Every Student Should Know".
Slide 1: Title slide
Slide 2: Why students need AI tools
Slide 3: Top 3 AI tools for studying
Slide 4: Top 2 AI tools for productivity
Slide 5: Getting started — your action plan
Use a clean, professional design.
```

**Step 2:** Review the generated presentation and:
- Change the design theme
- Edit at least 2 slides to improve accuracy or add your own perspective
- Add speaker notes to one slide using Copilot: *"Add speaker notes for slide 3"*

---

## Part 4: Copilot for Research & Summarisation (15 minutes)

Open copilot.microsoft.com and complete these tasks:

**Task 4.1 — Summarise a long document**
Paste a long article (500+ words) from any news site and ask:
*"Summarise this article in 5 bullet points. Identify the main argument and any limitations mentioned."*

**Task 4.2 — Compare two options**
Ask Copilot: *"Compare Python and JavaScript as a first programming language for a data science student. Present pros and cons in a table."*

**Task 4.3 — Generate a study plan**
*"Create a 4-week study plan for learning the basics of machine learning. I have 1 hour per day available. Include specific resources."*

Submit:

2. Excel: screenshot of the chart + your 3 documented insights
