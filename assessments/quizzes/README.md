**Q9.** Which of these is an example of regression (not classification)?
- A) Detecting whether an email is spam
- B) Predicting tomorrow's temperature
- C) Classifying an image as a cat or dog
- D) Identifying fraudulent transactions

**Answer: B**

---

**Q10.** Short answer: Explain the difference between supervised and unsupervised learning. Give one real-world example of each.

**Model answer:** Supervised learning uses labelled training data where the correct answer is known (e.g., training a model to detect spam emails using examples marked spam/not spam). Unsupervised learning finds patterns in unlabelled data without predefined correct answers (e.g., clustering customers by purchasing behaviour to identify market segments).

- B) Reduce the size of the model

- D) Speed up training by skipping irrelevant data

**Answer: A**

---

**Q10.** Short answer: In your own words, explain what a neural network is and why it is called "deep" learning when many layers are used.

**Model answer:** A neural network is a system of interconnected computational units (neurons) organised in layers that process information. The input layer receives data, hidden layers extract increasingly complex patterns, and the output layer produces predictions. It is called "deep" learning when there are many hidden layers — the depth allows the network to learn more abstract and complex features, which is what makes modern AI capable of recognising images, understanding language, and generating content.

**Answer: B**

---

**Q10.** Short answer: Explain how a Large Language Model generates text. What is it actually doing when it responds to your prompt?

**Model answer:** An LLM generates text by predicting the most likely next token based on all the tokens that came before it. It doesn't "know" facts or "understand" in the human sense — it has learned statistical patterns across vast amounts of text during training. When you send a prompt, the model repeatedly predicts the next most likely token, one at a time, until the response is complete. This is why outputs can seem fluent and confident even when they are factually wrong — the model is predicting plausible text, not retrieving verified facts.

## Module 05 — Prompt Engineering

**Q1.** What is "zero-shot prompting"?
- A) Asking AI a question with no context
- B) Asking AI to perform a task with no examples provided
- C) Sending an empty prompt
- D) Using AI without an internet connection

**Answer: B**

---

**Q2.** Which prompting technique asks the AI to reason step-by-step through a problem?
- A) Zero-shot prompting
- B) Role prompting
- C) Chain-of-thought prompting
- D) Few-shot prompting

**Answer: C**

---

**Q3.** In "few-shot prompting," what are the "shots"?
- A) API calls to the model
- B) Examples provided in the prompt to guide the AI's response
- C) Screenshots of previous conversations
- D) Short prompts under 10 words

**Answer: B**

---

**Q4.** What is "role prompting"?
- A) Asking AI to list the roles in a company
- B) Asking AI to adopt a specific persona or expertise
- C) Prompting AI using a structured template
- D) Giving AI a job title

**Answer: B**

---

**Q5.** Which of these is the STRONGEST prompt?
- A) "Write about productivity"
- B) "Write a blog post"
- C) "You are a productivity coach. Write a 300-word blog post for busy students on 3 study techniques. Use a motivational tone. Include one actionable tip per technique."
- D) "Tips for studying better please"

**Answer: C**

---

**Q6.** What is the main benefit of iterative prompting?
- A) It reduces the cost of API calls
- B) It allows you to gradually refine the AI output to match your exact needs
- C) It makes AI respond faster
- D) It prevents hallucinations

**Answer: B**

---

**Q7.** A "system prompt" in an LLM is:
- A) The user's question
- B) The AI's response
- C) Hidden instructions that configure the AI's behaviour before the conversation starts
- D) A technical error message

**Answer: C**

---

**Q8.** Which element is LEAST important in a well-crafted prompt?
- A) The specific task you want completed
- B) The format of the output you want
- C) The word count of your prompt
- D) Relevant context about who the audience is

**Answer: C**

---

**Q9.** When asking AI to explain a complex technical topic, what technique improves clarity?
- A) Using more technical jargon in your prompt
- B) Specifying the audience and their knowledge level
- C) Making the prompt as short as possible
- D) Using all capital letters

**Answer: B**

---

**Q10.** Short answer: Write an example of a weak prompt and then rewrite it as a strong prompt for the same task. Explain what you changed and why.

**Model answer (example):**
Weak: *"Help me write an email"*
Strong: *"You are a professional business communication expert. Draft a polite follow-up email to a recruiter I met at a career fair last Tuesday. I want to express interest in the Software Engineer Intern role we discussed and ask about next steps. Keep it under 120 words. Professional but not stiff."*
Changes: Added role context, specified the recipient and relationship, gave the specific purpose, defined the tone, set a length constraint. These elements remove ambiguity so the AI knows exactly what a good response looks like.
