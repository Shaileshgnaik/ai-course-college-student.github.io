# Assignment Specifications

This file contains the assignment brief for each of the 10 modules.

---

## Module 01 Assignment — AI Impact Reflection

**Type:** Written reflection
**Length:** 300–400 words
**Due:** Before Module 02 session

**Brief:**
Write a personal reflection on how AI will impact your chosen career over the next 5 years. Your reflection should:
- Name your intended career field
- Identify 2 specific ways AI is already being used in that field
- Predict 1 new AI capability that will emerge in the next 5 years
- Describe whether you see this as an opportunity or a threat — and why
- State one skill you want to develop to stay relevant

**Grading (10 marks):**
- Career field identified and context established (1)
- Two current AI applications described accurately (3)
- Future prediction is realistic and reasoned (2)
- Personal stance is clearly argued (2)
- Writing is clear and structured (2)

## Module 02 Assignment — Dataset Analysis

**Type:** Written analysis + diagram
**Length:** 300 words + one diagram or table
**Due:** Before Module 03 session

**Brief:**
Find any publicly available dataset on Kaggle or a similar source relevant to your field of study. Describe:
- The dataset: what it contains, how many rows/columns, where it came from
- The features (inputs) and the label (output), if applicable
- Which type of ML problem this is (classification, regression, clustering)
- What real-world decision this model could support
- One potential bias risk in the data

Include a screenshot of the dataset or a small sample table.

**Grading (10 marks):**
- Dataset described accurately (2)
- Features and label correctly identified (2)
- ML problem type correctly classified with reasoning (2)
- Real-world application is plausible and specific (2)
- Bias risk identified and explained (2)

## Module 03 Assignment — Neural Network Explainer

**Type:** Visual explainer (diagram + text)
**Length:** 1 page (A4) — diagram + 150–200 word explanation
**Due:** Before Module 04 session

**Brief:**
Create a one-page visual explainer showing how a Transformer model processes a sentence. Your explainer must show:
- The input (tokens)
- The attention mechanism (what it does, not how it works mathematically)
- The output generation (next token prediction)
- At least one label explaining why each step matters

You may draw by hand, use PowerPoint, or ask an AI to help you draft it — but the final explanation must be in your own words.

**Grading (10 marks):**
- Diagram is accurate and clearly labelled (4)
- Explanation is correct and in student's own words (3)
- Attention mechanism is correctly explained (2)
- Presentation is clear and readable (1)

## Module 04 Assignment — AI Study Assistant

**Type:** Practical + reflection
**Due:** Before Module 05 session

**Brief:**
Use ChatGPT, Microsoft Copilot, or another LLM to build a study assistant for one of your current subjects. Document the following:

1. **Prompts used** (at least 5) — write out the exact prompts
2. **AI outputs** — paste one full output as an example
3. **Your edits** — what did you change, and why?
4. **Hallucination check** — did the AI state anything factually wrong? How did you verify?
5. **Usefulness rating** (1-10) with a 2-sentence explanation

**Grading (10 marks):**
- 5 prompts documented (3)
- One full output included (1)
- Edits documented with reasons (3)
- Hallucination check completed (2)
- Usefulness rating with reasoning (1)

## Module 05 Assignment — Prompt Engineering Cheat Sheet

**Type:** Reference document
**Format:** 1–2 pages, designed to be printed or shared
**Due:** Before Module 06 session

**Brief:**
Create a "Prompt Engineering Cheat Sheet" for your discipline. It should include:
- The 5 key elements of a strong prompt (with a template)
- One example of a weak vs strong prompt for your field
- Examples of zero-shot, few-shot, and chain-of-thought prompting
- Your top 5 prompts for your area of study
- One "prompt anti-pattern" to avoid (with an example)

The cheat sheet should be designed so a classmate could pick it up and immediately use it.

**Grading (10 marks):**
- 5 key elements covered with template (2)
- Weak vs strong example is clear and relevant (2)
- Three techniques demonstrated correctly (3)
- Top 5 prompts are specific to the student's discipline (2)
- Anti-pattern is identified and explained (1)

## Module 06 Assignment — Internship Application Package

**Type:** Practical portfolio
**Due:** Before Module 07 session

**Brief:**
Use Microsoft Copilot to prepare a professional internship application package:
1. **Cover letter** (Word) — for a specific internship role in your field, under 200 words
2. **Skills summary** (Excel) — a table listing your skills, experience level (Beginner/Intermediate/Advanced), and evidence
3. **Introduction deck** (PowerPoint) — 3 slides: who you are, your top 3 skills, what you're looking for

For each document, note which parts Copilot generated and which parts you wrote or edited yourself.

**Grading (10 marks):**
- Cover letter is professional and appropriately targeted (3)
- Skills summary is complete and honest (3)
- Intro deck is clear and well-designed (2)
- AI/human contribution is documented (2)

## Module 07 Assignment — AI-Assisted Mini Project

**Type:** Working code + documentation
**Due:** Before Module 08 session

**Brief:**
Build a small Python project of your choice (suggestions: quiz app, to-do list, CSV data analyser, password generator, simple calculator with history). Requirements:
- Minimum 3 functions
- At least 50% of the code written or completed with AI assistance (GitHub Copilot, ChatGPT, etc.)
- Documented with docstrings (AI can help write these)
- A README explaining what the project does and how to run it

In a separate document, answer:
- What percentage of the code did AI generate?
- What did you have to fix or rewrite?
- Would you have been able to build this without AI assistance?

**Grading (10 marks):**
- Code works and meets the brief (4)
- At least 3 functions with docstrings (2)
- README is clear (1)
- Reflection on AI contribution is honest and thoughtful (3)

## Module 08 Assignment — AI Ethics Case Study

**Type:** Written analysis
**Length:** 500–600 words
**Due:** Before Module 09 session

**Brief:**
Choose a real-world AI controversy (suggestions: facial recognition bias, AI in hiring, deepfakes, AI in criminal sentencing, predictive policing, AI-generated academic work). Write an analysis covering:
1. What happened — describe the situation objectively
2. The ethical issues — what principles of responsible AI were violated?
3. Who was harmed and how
4. What should have been done differently
5. What should change going forward (policy, technical, or behavioural)

**Grading (10 marks):**
- Situation described accurately and objectively (2)
- Ethical issues identified with correct terminology (3)
- Harm analysis is specific and empathetic (2)
- Recommendations are realistic and reasoned (3)

**Type:** Industry brief
**Length:** 1 page (A4), approximately 400 words
**Due:** Before Module 10 session

**Brief:**
Create a "Future of AI in [Your Industry]" brief covering:

- Automation prediction is realistic and reasoned (2)

## Module 10 Assignment — AI Career Starter Pack

**Type:** Portfolio of 3 documents
**Due:** End of course

**Brief:**
Build your AI Career Starter Pack:

1. **Updated resume** — with a dedicated AI Skills section listing tools and techniques from the course (reviewed and edited, not just AI-generated)
2. **LinkedIn "About" section** — 200–300 words, rewritten with Copilot assistance, personalised and verified by you
3. **3-month learning roadmap** — specific goals, resources, and milestones for the 3 months after this course ends

For each document, note: what AI produced, what you edited, and why.

**Grading (10 marks):**
- Resume is professional with meaningful AI skills section (3)
- LinkedIn About is compelling and personalised (3)
- Roadmap has specific, achievable goals with named resources (3)
- AI/human contribution documented for each item (1)
