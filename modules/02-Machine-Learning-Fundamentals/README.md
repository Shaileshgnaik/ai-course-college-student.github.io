# Module 02 — Machine Learning Fundamentals

## Learning Objectives

By the end of this module students will:

- Understand what Machine Learning is and how it differs from traditional programming
- Identify the three types of machine learning
- Interpret basic ML concepts: features, labels, training, testing
- Recognize common ML algorithms and their use cases

## Topics Covered

### 1. What is Machine Learning?
### 2. Traditional Programming vs ML
### 3. Supervised Learning
### 4. Unsupervised Learning
### 5. Reinforcement Learning
### 6. Key Concepts: Features, Labels, Training, Testing, Overfitting
### 7. Common Algorithms Overview (Linear Regression, Decision Trees, K-Means)
### 8. ML in Practice — Real Industry Examples

## Demonstrations

- Training a simple classifier in Google Teachable Machine
- Visualizing decision boundaries
- Clustering demo with K-Means

## Hands-on Activities

- Build an image classifier using Teachable Machine (no code)
- Explore a dataset on Kaggle and identify features vs labels
- Predict house prices using a simple regression tool

## Assignment

Using a dataset of your choice, describe: the features, the label, which type of ML applies, and what real-world decision it could support.
