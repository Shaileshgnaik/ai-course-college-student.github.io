# Module 06 — Microsoft Copilot

## Learning Objectives

By the end of this module students will:

- Use Microsoft Copilot across Word, Excel, PowerPoint, and Teams
- Draft, edit, and summarize documents with Copilot in Word
- Analyse data and generate charts using Copilot in Excel
- Create presentations from prompts using Copilot in PowerPoint
- Summarize meetings and action items using Copilot in Teams

## Topics Covered

### 1. Introduction to Microsoft Copilot
### 2. Copilot in Word — Drafting and Editing
### 3. Copilot in Excel — Data Analysis and Formulas
### 4. Copilot in PowerPoint — Slide Creation
### 5. Copilot in Teams — Meeting Summaries and Chat
### 6. Copilot.microsoft.com — General AI Assistant
### 7. Responsible Use of Copilot in Academic Work

## Demonstrations

- Generate a report draft in Word from a bullet list
- Analyse a sales dataset in Excel using natural language
- Create a 10-slide deck in PowerPoint from a topic prompt
- Summarize a recorded Teams meeting

## Hands-on Activities

- Write a 500-word essay outline in Word using Copilot
- Analyse a provided dataset in Excel and generate 3 insights
- Build a presentation on any course topic using Copilot in PowerPoint

## Assignment

Use Microsoft Copilot to prepare a professional internship application package: cover letter (Word), skills summary (Excel), and intro deck (PowerPoint).
