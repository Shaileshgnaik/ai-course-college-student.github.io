A practical, hands-on AI course designed for college students across all disciplines. Covers AI fundamentals through to career readiness — with real tools, real prompts, and real projects.

---

## Course Modules

| # | Module | Topics |
|---|--------|--------|
| 01 | [Introduction to AI](modules/01-Introduction-to-AI/README.md) | AI history, types, real-world applications |
| 02 | [Machine Learning Fundamentals](modules/02-Machine-Learning-Fundamentals/README.md) | Supervised, unsupervised, reinforcement learning |
| 03 | [Deep Learning & Neural Networks](modules/03-Deep-Learning-Neural-Networks/README.md) | CNNs, RNNs, Transformers |
| 04 | [Generative AI & LLMs](modules/04-Generative-AI-and-LLMs/README.md) | GPT, Claude, Gemini, tokens, hallucinations |
| 05 | [Prompt Engineering](modules/05-Prompt-Engineering/README.md) | Zero-shot, few-shot, chain-of-thought, prompt patterns |
| 06 | [Microsoft Copilot](modules/06-Microsoft-Copilot/README.md) | Word, Excel, PowerPoint, Teams |
| 07 | [AI for Programming](modules/07-AI-for-Programming/README.md) | GitHub Copilot, Python, JavaScript |
| 08 | [Responsible AI](modules/08-Responsible-AI/README.md) | Ethics, bias, fairness, governance |
| 09 | [Future of AI](modules/09-Future-of-AI/README.md) | AGI, AI agents, industry disruption |
| 10 | [AI Careers & Roadmap](modules/10-AI-Careers-and-Roadmap/README.md) | Certifications, resume, interview prep |

## Getting Started

1. Browse the [Course Roadmap](COURSE-ROADMAP.md) for the full 10-week schedule
2. Start with [Module 01](modules/01-Introduction-to-AI/README.md)
3. Complete the exercises and assignments in each module folder
4. Work through the [Labs](labs/) for hands-on practice
5. Build one of the [Projects](projects/) as your capstone
